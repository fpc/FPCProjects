About the Fppkg Repository Server

Fppkg is the package system from Free Pascal. One of it's features is the ability to install packages from a central repository.
The goal of this website is to manage such a central repository.

To use this repository set 'remotemirrors' in your fppkg-configuration (fppkg.cfg) to http://fpc.cnoc.nl/repo/prod/mirrors.xml.

The sources of this site are available at https://svn.freepascal.org/svn/fpcprojects/fppkgreposerver/trunk.
