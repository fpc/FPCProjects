FPPKG Repository Server BuildAgent

Version 0.1, Joost van der Sluis (CNOC) -- 23 Oct 2017

Introduction

The Fppkg Repository Server Buildagent is a process that runs in the background awaiting for build-commands. It can contain several build- environments with different compiler-versions or targets. The process is accessible through an http-api.
