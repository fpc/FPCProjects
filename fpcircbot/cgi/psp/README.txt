 Pascal Server Pages provide easy way of web programming with Pascal
language and Free Pascal Compiler.

 Please see the documentation for more information.

--------------------------------------------------------------------
 
 This package includes:

 bin:

    conf - compiled configuration tools and configuration files for
           Windows operating system.

    examples - compiled examples for Windows operating system.


 doc - Pascal Server Pages documentation.

 src:

    conf - sources of configuration tools and configuration files.

    examples - example programs sources.

    templates - simple application templates.

    units - Pascal Server Pages units.
